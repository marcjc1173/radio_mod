Radio Mod v1.0.0 - Simple Installation
=====================================

ONE-CLICK INSTALLATION:
=======================

Just double-click SETUP.bat and you're done!

The setup script will:
✓ Install the mod automatically
✓ Extract portable tools (no system installation needed!)
✓ Configure everything for you

MANUAL INSTALLATION (if SETUP.bat doesn't work):
================================================

1. Copy radiomod-1.0.0.jar to:
   %appdata%\.minecraft\mods

2. If tools.zip is included, extract it to:
   %appdata%\.minecraft\radiomod\tools

REQUIREMENTS:
=============

- Minecraft 1.20.4
- Minecraft Forge 49.2.4
- Java 17 or higher

That's it! No Python, no ffmpeg installation needed if tools.zip is included.

USAGE:
======

1. Launch Minecraft with Forge
2. Craft a Radio (check creative menu)
3. Place it in your world
4. Right-click to open the music player
5. Paste a YouTube URL and click Play!
